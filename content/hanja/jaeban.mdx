<!-- step:cinematic-hook -->
계약서에 이런 문장이 있었다.
"제반 비용은 갑이 부담한다."
'제반'이 무슨 뜻인지 몰라 계약서를 대충 넘긴 적이 있는가?
'제반(諸般)'은 '모든' 또는 '여러 가지'를 뜻한다.

<!-- step:concept -->
제반(諸般)은 한자 '모두 제(諸)' + '가지/종류 반(般)'으로 이루어진 한자어다.
'여러 종류의 모든 것'이라는 의미로, 특정 범주 안의 모든 것을 가리킨다.
계약서, 법령, 공문서, 규정 등 법률적/행정적 문서에서 빈번히 사용된다.
일상 대화에서는 거의 쓰이지 않고 공식 문서에서만 나타난다.

<!-- step:example -->
제반이 쓰이는 실생활 예시:
- 제반 비용 → 모든 비용
- 제반 서류를 구비하시오 → 필요한 모든 서류를 갖추시오
- 제반 사항을 검토하다 → 모든 사항을 검토하다
- 제반 절차를 이행하다 → 모든 절차를 이행하다
- 제반 여건을 고려하다 → 여러 가지 여건을 고려하다

<!-- step:fact -->
'제(諸)'라는 한자는 '모두', '여러', '다수'를 의미하는 접두사로 자주 쓰인다.
제도(諸島) = 여러 섬, 제국(諸國) = 여러 나라, 제왕(諸王) = 여러 왕 등에도 사용된다.
'제반'은 조선시대 공문서부터 현대 법률 문서까지 일관되게 쓰이는 유서 깊은 한자어다.

<!-- step:vs -->
'제반'과 유사한 한자어 비교:
- 제반(諸般) = 여러 가지 모든 것 (포괄적 나열)
- 일체(一切) = 모든 것, 전부 (완전한 전부)
- 각종(各種) = 여러 종류 (다양한 종류 강조)
- 전반(全般) = 전체적인 모든 것 (전체를 아우름)
- 모든 → 일상 대화에서 쓰는 순우리말

<!-- step:quiz -->
근로계약서에 "제반 복리후생은 회사 내규에 따른다"라는 문장이 있다. '제반'의 의미는?
A) 일부
B) 모든, 여러 가지
C) 선택적인
D) 임시적인
ANSWER:B

<!-- step:memory-tip -->
諸(제) = 모두, 여러 / 般(반) = 가지, 종류
'제주도(濟州道)'가 아닌 '諸' — 여러 종류를 아우른다는 뜻!
제반 = 여러 가지 모든 것 = 공문서의 '전부'

<!-- step:summary -->
1. 제반(諸般) = 모든, 여러 가지 — '모두 제(諸)' + '가지 반(般)'
2. 계약서, 법령, 공문서에서 '모든 비용', '모든 서류' 뜻으로 사용
3. 제반 = 해당 범주의 모든 것 — 'all-inclusive'의 한자어 버전
